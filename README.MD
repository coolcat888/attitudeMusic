# ATM态度音乐

## 环境
  node
  mongodb

## 启动
```
  新建终端1
  cd attitudemusic
  npm i
  npm run serve
  新建终端2
  cd attitudeBackground
  npm i
  nodemon app.js
  新建终端3
  cd NeteaseCloudMusicApi-master
  npm i
  nodemon app.js
```
